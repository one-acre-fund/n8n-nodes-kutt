# n8n-nodes-kutt

![logo](./docs/logo.svg)

This is an n8n community node to interact with the [Kutt](https://kutt.it/) URL shortening service.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/reference/license/) workflow automation platform.
